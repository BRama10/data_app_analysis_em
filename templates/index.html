How it works?

The application connects to your Google Drive and extracts the data sheets that need to be analyzed. Once extracted, data formatting and regularizing occurs, which includes converting files to the correct format. Then data cleaning and filtering is done, including replacing null values, eliminating useless columns & rows. Finally, a database is initialized and populated to allow for queries. The data is also used for visualizations and predictions, as well as assists in manual observations (such as calling specific custom-built queries).
